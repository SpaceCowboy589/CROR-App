To include the official CROR PDF inside the app (fully offline):

1) Download the official PDF from Transport Canada:
   https://tc.canada.ca/sites/default/files/2022-05/canadian-rail-operating-rules-may-9-2022.pdf

2) Rename the downloaded file to exactly: cror.pdf

3) Upload the file to the project's assets/ folder in Replit:
   - Click the three-dot menu next to the 'assets' folder → Upload file → select cror.pdf

4) After uploading, run the app. The Reader tab will use the local PDF for offline reading.
